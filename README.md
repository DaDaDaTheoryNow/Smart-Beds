## Technologies
> In this project, I created a Smart Beds App for monitoring your garden beds

**Server**
- FastApi (https://hackaton15.onrender.com/docs)
- Pyhton (https://github.com/kirill221100/hackathon151223)
- SQLite

**Client**
- Flutter
- GetX
- RestApi

**Architecture**
- Layered Architecture
